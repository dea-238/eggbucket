<h2>To set up the project locally, follow these steps:</h2>
<h3>Frontend</h3>
<h4>cd frontend</h4>
<h4>npm install</h4>
<h4>npm run build</h4>
<h4>cd ..</h4>
<h3>Backend</h3>
<h4>cd backend</h4>
<h4>npm install</h4>
<h4>npm start</h4>


<h3>To start frontend only</h3>
<h4>cd frontend</h4>
<h4>npm install</h4>
<h4>Go to dashboard.js and comment out the useEffect to prevent auth for dev</h4>
<h4>npm start</h4>
